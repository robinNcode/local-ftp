# Local FTP Server

A simple FTP server that allows you to upload and download files from your local network.

## Features

- Upload and download files
- Delete files
- List files
- Download as a ZIP archive
- Download as a CSV file
- Download as a JSON file
- Download as a PDF file
- Download as a TXT file

## Usage

### Backend

1. Clone the repository:

```bash
git clone https://github.com/robin-tian/local-ftp.git
```

2. Build the backend:

```bash
cd backend
go build -o local-ftp
```

3. Run the backend:

```bash
./local-ftp
```

### Frontend

1. Clone the repository:

```bash
git clone https://github.com/robin-tian/local-ftp.git
```

2. Install dependencies:

```bash
cd frontend
npm install
```

3. Build the frontend:

```bash
npm run build
```

4. Run the frontend:

```bash
npm run dev
```
